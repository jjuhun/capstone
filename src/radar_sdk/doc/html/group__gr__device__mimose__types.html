This includes enumerations that can be used for the components of the Mimose device configuration structure ifx_Mimose_Config_t.

enum ifx_Mimose_Channel_t

Defines the channel type by the TX and RX configuration (expressed as a pair / combination of the two TX and RX).

This enum defines the channel type to be used. The ATR22 has 2 directional Antenna pairs and the channel defines the TX and RX configuration expressed as a pair/combination.
